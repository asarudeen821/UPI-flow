## 🚀 How to Use

### For Users

1. **Start the application** (if not already running):
   ```bash
   # Terminal 1 - Backend
   cd backend
   npm run dev

   # Terminal 2 - Frontend  
   cd frontend
   npm run dev
   ```

2. **Navigate to Transactions**:
   - Go to http://localhost:5174/transactions
   - View your transaction history

3. **Download a Receipt**:
   - Click the **"Receipt"** button on any transaction
   - Click the **"Download PDF"** button (blue button)
   - Wait 1-2 seconds for PDF generation
   - The receipt will download automatically as `receipt_<transaction_id>.pdf`

## 🛠️ Troubleshooting

### PDF not downloading?
1. Check browser pop-up blocker settings
2. Ensure JavaScript is enabled
3. Try a different browser

### PDF looks distorted?
1. Clear browser cache
2. Refresh the page
3. Try downloading again

### Backend API not working?
1. Ensure backend server is running on port 3000
2. Check MongoDB connection
3. Verify transaction exists in database

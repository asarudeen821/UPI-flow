## User Experience

### How to View Receipt

1. **Navigate to Transaction History**
   - Go to `/transactions` page

2. **Find Your Transaction**
   - Use search or filters to locate the transaction
   - Switch between List and Timeline view if needed

3. **Click Receipt Button**
   - **List View:** Click the "Receipt" button next to the amount
   - **Timeline View:** Click the receipt icon button next to the amount

4. **View Receipt Details**
   - Modal opens with full transaction details
   - All information is clearly organized

5. **Take Actions**
   - **Copy Transaction ID** - Click the copy icon or "Copy ID" button
   - **Copy UPI ID** - Click copy icon next to UPI ID
   - **Copy Mobile Number** - Click copy icon next to mobile number
   - **Download Receipt** - Click "Download" button to save as `.txt` file
   - **Close Modal** - Click X button or outside modal

### Download Functionality

The download feature creates a text file with formatted receipt data:

```javascript
const handleDownload = () => {
  const receiptText = `
PAYMENT RECEIPT
===============

Transaction ID: ${transaction.transaction_id}
Status: ${transaction.status.toUpperCase()}
Date: ${new Date(transaction.created_date).toLocaleString('en-IN')}

Amount: ₹${transaction.amount}

Recipient: ${transaction.recipient_name}
UPI ID: ${transaction.upi_id}
Mobile: ${transaction.mobile_number}

Note: ${transaction.note || 'N/A'}

---
Generated by PayApp
  `.trim()

  const blob = new Blob([receiptText], { type: 'text/plain' })
  const url = URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = `receipt_${transaction.transaction_id}.txt`
  a.click()
  URL.revokeObjectURL(url)
}
```

### Copy to Clipboard

```javascript
const handleCopy = (text, field) => {
  navigator.clipboard.writeText(text)
  setCopied(field)
  setTimeout(() => setCopied(null), 2000)
}
```

Visual feedback shows "✓" (Check icon) for 2 seconds after copying.
